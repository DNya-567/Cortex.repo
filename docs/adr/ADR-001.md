# ADR-001: Use Arrow Functions for Callbacks

## Status
accepted

## Context
We needed a way to handle asynchronous operations in our JavaScript codebase. Arrow functions provide lexical `this` binding which prevents common bugs in callback handlers.

## Decision
All callback functions should be written as arrow functions unless explicitly needed otherwise.

## Consequences
- Easier to reason about `this` context
- More consistent code style
- Potential issues if external libraries expect function expressions with their own `this`

## Affected Files
- test-codebase/auth.js
- test-codebase/userRepository.js
